Shift Cipher
Author: Sam Martinez
Shift Cipher gives you the ability to type in a phrase and have it shift based on a value you input between 1-26. 



Installation

Install Shift Cipher by cloning this repository:

https://github.com/sammartinez/sam_martinez_cipher.git

- Use MAMP and point the Root Folder under Preferences/Web Server
- Start Server (Also verify you have the requirements needed in order to run MAMP on your client machine. 
- Once the server is running, open WebStart page, then import the database from the following folder: 

bookstore/sites/sites/db-backup 

- Navigate to localhost:8888

-Enjoy
